# Changelog Rule — MANDATORY

**CRITICAL: Update CHANGELOG.md after EVERY code change.**

Format: `## YYYY-MM-DD` section, one line per change at TOP of today's section.
Prefixes: `FIX:` / `FEATURE:` / `REFACTOR:` / `CHORE:` — present tense ("Add", "Fix", not "Added").

Workflow: code change → clippy/fmt → **update CHANGELOG.md** → done.
